notifier: (id)  notifier 
implementation

If error param is non-nil, report the given error via OOCastErrorNotification's userInfo dictionary.

Parameters
errorif non-nill will be passed in userInfo dictionary as"error":error} and will not be over-written by any "error" key-value in extras.
extrasif non-nil will be used as the base for the userInfo dictionary.
notifierbecomes the 'object' notification parameter.
