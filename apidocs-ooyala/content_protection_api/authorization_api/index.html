Player Authorization API

version v3 & v4

http://player.ooyala.com/sas/player_api

Overview

The Ooyala Player Authorization API is a set of APIs (application programming interfaces) for defining and managing content protection. The Authorization API is used by the player to verify the authorization to entitlements at playback and when required issue licenses or other credentials for DRM (Digital Rights Management) systems. In the majority of cases, authorization simply occurs, without any programming.

The Authorization API has two versions: one for Player v3 and one for Player v4.

Player v3 Authorization API

When the caller makes an authorization request, the Authorization API responds to the caller with a JSON array indicating the authorization status for each of the embed codes included in the list in question.

/v1/authorization/embed_code/{pcode}/{embed_codes} get
