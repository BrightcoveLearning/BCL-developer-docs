Ooyala IQ SDK for Roku

This library allows you to report video events (content started, content paused, and so on) and certain user metrics (platform, OS, and so on) from your Roku channels to Ooyala IQ. It provides a very simple and easy to use API through the IQ object.

Roku requirements

The minimum Roku OS version compatible with this library is v7.

Getting started

See the Getting Started tutorial for information about how to use the library.
