Introduction

The Video Cloud architecture has an API generally for each module in Studio, and sometimes for specific functionality within a module. Therefore, the operations enabled by the Backlot API correspond to operations within different Brightcove API. This document will help you understand these mappings.

Authentication

Authentication of Brightcove API requests is handled slightly differently than that for the Ooyala REST APIs.
