Video Cloud Studio Features Not Available in Backlot

This topic outlines some of the features that are available in Video Cloud Studio that aren't available in Backlot.

This topic covers some of the more popular features that Video Cloud publishers implement that aren't available in Backlot. This list is not comprehensive. For each feature, a brief description and links to documentation or a video where you can learn more are provided.

  • Playlists
  • Retranscoding videos
  • Custom metadata
  • Media sharing
  • Custom reports
  • Video experiences
  • Marketing automation platform integration

Playlists

A playlist is a collection of videos that are grouped together for playback in a Brightcove Player. Two types of playlists can be created using the Media module, Manual and Smart. With a Manual playlist, you select the videos that are in the playlist. With Smart playlists, videos are automatically added to the playlist based upon the tags that associated with videos.

Published playlists appear next to the player and allow the user to select which videos they want to watch.

  • Playlist (Vertical) - Playlist displays on the right side of the player.
  • Playlist (Horizontal) - Playlist displays on the bottom of the player.

Retranscoding videos

When videos are uploaded using the Upload module, the selected ingest profile (similar to Backlot's processing profile) determines the quality and number of renditions that are created. The Media module can be used to retranscode existing videos using a different ingest profile. Retranscoding will create a new set of renditions for the video. Retranscoding saves you from having re-upload the video, re-enter the video metadata and any existing video analytics are not lost.

Custom metadata

Custom metadata can be used to add additional metadata fields to your videos. In Backlot, custom metadata can be added for each video. With Video Cloud, custom fields are added as part of your account settings. This makes these fields available to all videos that are uploaded. After the custom fields have been added, the fields can be assigned values using the Media module.

Note: Custom metadata is available only for Video Cloud Pro and Enterprise customers. If you are interested in upgrading your Video Cloud account, please contact Brightcove for more information.

Media sharing

Media sharing allows publishers to upload videos once and then share the videos to other accounts. The accounts can be your own, corporate accounts or accounts of other publishers. The shared videos can then be used independently in the other Video Cloud accounts. Media sharing is typically used when publishers want to keep a master account of video content and then share videos out to other divisions or subsidiaries of the organization.

Note: Media sharing is available only for Video Cloud Pro and Enterprise customers. If you are interested in upgrading your Video Cloud account, please contact Brightcove for more information.

Custom reports

While both Backlot and Video Cloud offer various types of video and player analytics in the UI, Video Cloud offers the ability to create and schedule custom analytics reports. This allows you to create reports with the specific data fields you want to see. Reports can be filtered to return a subset of your video data and can be scheduled to run on a daily, weekly or monthly basis.

Note: Custom reports are available only for Video Cloud Pro and Enterprise customers. If you are interested in upgrading your Video Cloud account, please contact Brightcove for more information.

Marketing automation platform integration

Brightcove Players can be configured to track video engagement data for videos. This can be used to track viewing behavior of specific viewers. Viewing data is stored in Video Cloud and can optionally be synchronized to popular marketing automation platforms such as Eloqua, Marketo, HubSpot and Salesforce.
