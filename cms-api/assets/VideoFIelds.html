whether video is enabled for offline viewing (DRM-packaged videos only)

Size range: ..150

state determines whether the video is playable or not

Allowed values: "ACTIVE", "INACTIVE"

array of tags - note that tags may not contain commas

map of fieldname-value pairs; values have a maximum length of 1024 single-byte characters Note: be sure to use the internal name for the field, not the display name
