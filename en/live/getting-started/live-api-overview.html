Overview: Brightcove Live API

In this topic, you will learn what the Policy API is for and how to use it. The Live API is a REST-based API that allows you to create and manage live streaming events, with or without server-side ad insertion (SSAI). It also allows you to create VOD assets from clips taken from a live stream.

Base URL

The base URL for the Live API is:

https://api.bcovlive.io

Headers

All requests are authenticated using an API key that will be provided to you when your account is set up. The key is passed in an X-API-KEY header. A Content-Type header is also required:

X-API-KEY : YOUR_APIKey
Content-Type: application/json

Endpoints and operations

The main operations for the Live API are creating and managing live streams, and generate VOD clips from live streams.
