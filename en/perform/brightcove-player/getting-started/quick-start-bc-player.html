Quick Start to Brightcove Player

In this quick start you will perform the following tasks:

  • Create a player and assign it a video using Studio
  • Assign video to the player
  • Publish the player
  • Use the iframe player implementation
  • Use the In-page embed implementation
  • Programmatically play video

Create player

To create a new player, follow these steps.

  1. Login to Video Cloud Studio at https://studio.brightcove.com/products/videocloud/players/.
  2. The current Video Cloud account name will appear in the upper right corner of the page. If you have multiple accounts, click the current user - current account link and select the account you would like to create the player in.
  3. Click the Players link in the navigation header.
  4. Click the New Player button.
  5. Enter a Name and Short Description for the player. For this Quick Start, the player name will be Sample Player.
  6. Confirm that the player appears in the list of players.

Assign video to player

The Video Content properties are used to assign video content to the player. The video can reside in a Video Cloud account or in another location. Note that the properties that are available will change based upon how the video content is associated to the player.

To configure the video content properties, follow these steps.

  1. Click the link for the Sample Player to open the player properties page.
  2. Locate the Video Content section and click Edit.
  3. Select Video ID for the method for associating video content to the player. This indicates that a Video Cloud video will be used with the player.
  4. For Video ID, enter the Video Cloud video ID for the video you want to publish. You can get the video ID by opening the Media module and locating the video. The video ID will appear on the right side of the page next to the video thumbnail.
  5. Do not set any dimensions for the video as this will be done via CSS in the page.
  6. Click Save.

For more information on the Video Content properties, see Configuring the Video Content Properties for a Player.

Publish player

Publishing the player will push the changes to the published player. For more information on preview players, published players and player publishing code, see Generating Player Publishing Code.

To publish the player, follow these steps.

  1. Click Publish and then Publish again to confirm the player publish.
  2. Confirm that the publish was successful.
  3. Click Embed Code & URL and click the Published Player link. The Published Player Embed Code dialog will display.
  4. Copy all three of the code implementation type, Player URL, iframe Embed Code and In-page Embed Code to a text file for later use.
  5. The Player URL can be used to test player/video functionality quickly in a browser. Open a new browser tab and paste in the Player URL to view the published player. The player should occupy the entire width of the browser page.

Note: If curious, you can see "under the hood" of the player configuration by changing index.html at the end of the URL to config.json. You will see a JSON object detailing the current configuration of the player you just created.

Use iframe

Note: To use the iframe implementation you will need to use a web server to browse the page because of iframe requirements. Simply viewing the page locally will not work.

  1. Create an HTML page and paste the iframe Embed Code code from step 15 into the body.
  2. When you browse this page you will see the video displays in the default size of 300px wide and 150px high. Depending on the actual width and height of your video you may see letter boxing or column boxing.
  3. To set the size of the iframe to a different size you can use CSS. The following uses a CSS element selector you change the size of the iframe, and the video will fill the frame size.
    4.   <style>
   iframe {
      height: 344px; 
      width: 610px;
    }
  </style>
  4. Save your changes and refresh your page in the browser to see the video at the new size.
  5. To see the built-in responsive sizing capabilities of the iframe player implementation change both the height and width in the iframe selector to 100%.
    6.   <style>
   iframe {
      height: 100%; 
      width: 100%;
    }
  </style>
  6. Now surround the iframe tag in a <div> tag block with a class of videoFrame.
    7. <div class="videoFrame">
  <iframe ...>
</div>
  7. Add a class selector style for the newly create <div> element.
    8. .videoFrame {
   height:56%; 
   width:100%;    
}
  8. Save your changes and refresh your page in the browser. As you resize the browser you will see the iframe player implementation fill the changing size of the <div> element.

Note: You will see letter boxing and pillar boxing as you resize the browser. To see how to avoid this, the Responsive Sizing for the Brightcove Player document details the solution.

Use In-page embed

In this section you will use the In-page embed code implementation

  1. Create an HTML page and paste the In-page Embed Code code (also called embed_in_page) from step 15 into the body.
  2. When you browse this page you will see the video displays in the default size of 300px wide and 150px high. Depending on the actual width and height of your video you may see letter boxing or column boxing.
  3. To set the size of the iframe to a different size you can use CSS. The following uses a CSS class selector you change the size of the player.
    4. <style>
  .videojs {
    height: 344px;
    width: 610px;
  }
</style>
  4. Save your changes and refresh your page in the browser to see the video at the new size.

Programmatically play video

There is a rich API to use with Brightcove Player. In this section you will use the play() method to programmatically start the video playing.

  1. Add an id attribute to the video tag with a value of myPlayerID.

  2. Just above the closing body tag, insert a script block.

  3. In the script block, use the ready() method to bind the player's ready event to an anonymous event handler function.

  4. Create a variable local to the function, named myPlayer, that assigns the player instance, referenced as this, to that variable.

  5. Use the player's play() method to start the video.

  6. Save your changes and refresh your page in the browser to see the video automatically start playing.
