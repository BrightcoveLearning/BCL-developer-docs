Size the Player

Note: the Brightcove player is currently in beta and requires some developer skills to implement. If you are interested in participating in the beta, please contact your account manager.

There are several ways of sizing the player. The most important factor in which method you choose is the type of player combinations you will be using, that is iframe only, in-page only, or both iframe and in-page. The different ways of sizing the player are described below.

iframe and in-page

For both the iframe and in-page embed, the size can be supplied in the media configuration option of the player. For in-page embeds, the player will use that size specifically. For iframe embeds, the aspect ratio will be calculated based on the supplied values, if they are in pixels (pixel value is assumed).

To create a player and set the size, you could do something like the following:

{
  "name": "MySamplePlayer",
  "configuration": {
    "media": {
      "height": "720px",
           "width": "1280px",
      "sources": [{
        "src":"http://solutions.brightcove.com/bcls/assets/videos/Tiger.mp4",
        "type":"video/mp4"
      }]
    }
  }
}

This will make the in-page embed a 720x1280 sized player. The iframe embed would calculate a 16:9 aspect ratio to use based on this input and then responsively size the player.

Stylesheet

For both the iframe and in-page embed, a custom stylesheet can be supplied. This stylesheet is given to the player in the stylesheets configuration, and since it is in the configuration would be applied to both iframe and in-page players.

{
  "name": "MySamplePlayer",
  "configuration": {
    "media": {
      "sources": [{
        "src":"http://solutions.brightcove.com/bcls/assets/videos/Tiger.mp4",
        "type":"video/mp4"
      }]
    },
    "stylesheets": ["http://example.com/styles/custom.css"]
  }
}

In the stylesheet, the player can be sized via something like:

.video-js {
  height: 720px;
  width: 1280px;
}

In-page solutions

It is possible to use sizing solutions that will work only for the in-page embed. You can supply width and height attributes on the video element directly to style the video.

<video
  data-account="123"
  data-player="123"
  data-embed="123"
  data-id="123"
  class="video-js" controls
  width="1280" height="720"></video>

You can move the width and height to a stylesheet for the video-js class that is applied to the player, as shown:

<style type="text/css">
  .video-js {
    height: 720px;
    width: 1280px
  }
</style>
