Working with Media Content

Video Cloud customers have access to their media data and metadata from Video Cloud Studio. Perform customers will supply the URL for their media content.

Video Cloud customers

As a Video Cloud customer, you can access your media data stored in Video Cloud Studio.

Retrieving media data

You can retrieve your media data from Video Cloud Studio in one of two ways:

  • Using the Media API:

    The Media API is a RESTful API that lets you interact with your Video Cloud media library. For details about using this API, see the Media API Documentation page.

  • Using the Catalog class:

    The Catalog provides access to Video Cloud videos and playlists. This component's methods allow retrieval of videos and playlists by ID and/or ReferenceID. The Catalog will make the URL requests and parse the returned data.

Geo Restrictions

When you use Brightcove's CMS API to add geographic filtering properties to your videos, you can control which countries they can (or cannot) be viewed in. The Brightcove Player SDK for Android supports geo-restricted videos. Learn how to add geo-filtering using the CMS API.
