Brightcove Player SDK for iOS: Previous SDK information

This topic shows the previous SDK and plugin versions for the Native Player SDK for iOS.

The following links for the previous version of the Brightcove Native Player SDK for iOS is for reference only. It is recommended to use the latest version of the SDK along with the plugin integrations.
