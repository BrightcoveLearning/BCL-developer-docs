This document is the comprehensive collection of errors for Brightcove Player. Note that errors are thrown from a number of different plugins.

Types of errors

There are five main types of errors which are briefly described in this section. Then each type will be detailed in a later section of this document.

  1. Standard errors dispatched from the Errors Plugin
    • These error names start with MEDIA_ERR
    • Each specific error has a specified positive integer ID number
    • See the Standard errors section of the Display Error Messages Plugin document for complete details.
  2. Custom errors defined by Brightcove and dispatched from the Errors Plugin
    • These error names start with PLAYER_ERR
    • Some of these errors have a specified negative integer ID number
    • See the Custom errors section of the Display Error Messages Plugin document for complete details.
  3. Custom errors that are user defined and dispatched by the Errors Plugin
    • These error names are defined by by the user, for example AGE_GATE_ERROR
    • See the Adding custom errors section of the Display Error Messages Plugin document for complete details.
  4. Errors dispatched from the Video Cloud catalog functionality
    • These error names start with VIDEO_CLOUD_ERR
    • See the Catalog errors section of the Player Catalog document for complete details.
