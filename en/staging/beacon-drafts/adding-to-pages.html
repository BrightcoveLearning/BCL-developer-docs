DRAFT - Adding a Carousel to an Existing Page

In this topic, you will learn how to add a group of videos in a carousel to a Brightcove Beacon Page.

Overview

The Brightcove Beacon Page is where users will select and watch your video content. Carousels display a group of videos or playlists that you have defined in Video Cloud Studio. Using the Brightcove Beacon Administrative Console, you can add carousels to an existing page.

Adding a carousel

To add a carousel to an existing Page, follow these steps:

  1. Login to Brightcove Beacon.

  2. In the top navigation, select Pages.

  3. You can use the Add new page button to create a new page. Since pages should already be set up for you, we will walk through the process of adding a carousel to an existing page. Select the page that you want to update. Here, we are selecting the link for the Home page.

  4. Notice that the Basic Data is already populated.

  5. Select the Add Block button.

  6. Choose the type of block that you want. In this case, we are choosing the list-block. Select the Add Block button associated with the block that you want.

  7. Choose the type of layout that you want, whether it is Featured, Carousel or Grid. In this example, we are selecting the Carousel Landscape Labels In layout.

  8. Now that you have the container defined for the carousel, you are ready to add content. From the list-block item, select the Add Playlist button.

  9. You create playlists using Video Cloud Studio. They will appear here once you sync to your Video Cloud account. Choose a playlist. In this example, we are selecting the Add Playlist button associated with the Humor playlist.

Rights & Scheduling

The Rights & Scheduling section is where you set the publish status and select the payment type for your playlist.

  1. Select the Rights & Scheduling tab.

  2. Select a status:

    • Published - Your playlist is published to your apps.
    • Unpublished - Your playlist is not published.
    • Pre-Prod - Your playlist is set to pre-production status.

  3. In the Availability Rights section, select a payment method:

    • No Payment
    • Stripe
    • Roku Store
    • Apple Store
    • Google Play Store
    • Amazon

  4. When No Payment is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.

    To add or edit geographic locations, navigate to Settings and then Locations.

  5. When a payment option is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.
    Price Yes Enter the currency amount.
    Currency Yes The currency type. Use the Add Currency button to add other currencies.
    Vendor ID Yes Enter your vendor ID. In this example, it is your Amazon ID.

    To add or edit geographic locations, navigate to Settings and then Locations.
