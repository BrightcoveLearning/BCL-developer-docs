DRAFT - Brightcove Beacon Advertisement Configuration

In this topic, you will learn how to configure advertising using the Advertisement tab for Brightcove Beacon.

Overview

An advertisement is associated with either a live or VOD ad configuration ID. You will create the ad configuration in Video Cloud Studio. For details, see the Video Cloud Ad Configuration ID document.

Once you have created an advertisement, then you will be able to select it when creating a channel, movie, episode or playlist.

Advertisement

To create an advertisement, follow these steps:

  1. Login to Brightcove Beacon.
  2. In the top navigation, select Advertisement.

  3. Select the Add New Configuration button.

  4. Here there is only one tab for Basic Data.

  5. Enter values for the following fields:

    Field Required Notes
    Name Yes A descriptive name for your advertisement definition. This could be your campaign name.
    Provider Yes The value is Brightcove SSAI. This allows you to stitch ads into your stream using Brightcove Server-Side Ad Insertion (SSAI).
    Live Ad Config ID No The ID for your live ad configuration created in Video Cloud. Either this or the VOD Ad Config ID is required.
    VOD Ad Config ID No The ID for your VOD ad configuration created in Video Cloud. Either this or the live Ad Config ID is required.

    You will create the ad configuration in Video Cloud Studio. For details, see the Video Cloud Ad Configuration ID document.
  6. Select the Create New Advertisement button.
