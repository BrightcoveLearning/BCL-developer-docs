Cut, Paste, and Go! Using Pre-roll Ads with Brightcove Player

Introduction

Want to use pre-rolls with your short-form video content? Follow these steps and cut-and-paste the example code to get up and running in no time.

Prerequisites

  1. An ad tag: If you’re using DFP or any other VAST-compliant ad server or demand source then you likely already have easy access to an ad tag, also known as the URL to the ad server, or serverURL. 
    https://ads.brightcove.com/ads?vc=1&dur=15
  2. A video: Let’s assume use of an HLS, MP4 or a Brightcove Video Cloud video. In this example an HLS source URL is used. 
    https://devstreaming-cdn.apple.com/videos/wwdc/2017/811evqx1dsujdj5222/811/hls_vod_mvp.m3u8
  3. A Brightcove Video Cloud or Brightcove Player account or trial account: Login to http://studio.brightcove.com to set up an account, or get your current account ID. 
    3723496277001
You will need to replace these prerequisite values from your account/environment before running the code.

The code

And here’s the code. The values you need to replace in the code are highlighted in yellow.

<html>

<head>
  <title>Pre-roll Demo</title>
</head>

<body>
  <!-- The Brightcove Player in-page embed code. You can copy and paste this (but remember to replace the account number) -->
  <div id="playerDiv" style="width:512px;height:288px">
    <video data-account="3723496277001"
      id="myPlayerID"
      style="width:inherit;height:inherit"
      data-player="default"
      data-embed="default"
      class="video-js"
      controls></video>

    <!-- Add a data-video-id attribute to the video element if you want to use a Video Cloud video ID or -->
    <!-- reference ID instead, like this: data-video-id="ref:big_buck_bunny_dd" -->
  </div>

  <!-- The Brightcove Player JavaScript for the default player on the account. You can use any player in your Brightcove account but this one is an easy place to start that's always defined. -->
  <script src="//players.brightcove.net/3723496277001/default_default/index.js"></script>

  <!-- This next line is the stylesheet for the Brightcove IMA ad plugin -->
  <link rel="stylesheet" type="text/css" href="//players.brightcove.net/videojs-ima3/2/videojs.ima3.min.css" />

  <!-- The JavaScript library for the Brightcove IMA ad plugin -->
  <script src="//players.brightcove.net/videojs-ima3/2/videojs.ima3.min.js"></script>
  <script type="text/javascript">
    videojs('myPlayerID').ready(function(){
      var myPlayer = this;
    myPlayer.ima3({
      // only use HTML5 mode
      adTechOrder: ["html5"],
      // you can increase or decrease the max amount of time to wait for an ad
      timeout: 5000,
      // this is where your ad tag goes
      serverUrl: "https://ads.brightcove.com/ads?vc=1&dur=15"
    });
     // now load the video - you can also comment this part out and use a video id
     // (supplied in the video element above as data-video-id)
    myPlayer.src({
      src: "https://devstreaming-cdn.apple.com/videos/wwdc/2017/811evqx1dsujdj5222/811/hls_vod_mvp.m3u8",
      type: "application/mpegURL"
    });
     // play the video
    myPlayer.play();
  });
  </script>
</body>

</html>
