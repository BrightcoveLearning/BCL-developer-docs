Multi-Channel Audio Processing

Return to the Documentation Index

Note: these topics apply only to Ooyala customers recently migrated to Video Cloud.

Introduction

Multi-channel audio in your source videos can be preserved in your Video Cloud video renditions. To do this, you need to have some special renditions for ingest profiles added to your account. You can have this done by contacting Brightcove Support. Support will also help you create an ingest profile using these renditins, which you can then use when you are adding multi-channel audio videos.
