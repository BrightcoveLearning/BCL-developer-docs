Multi-Channel Audio Processing

Return to the Documentation Index

Note: these topics apply only to Ooyala customers recently migrated to Video Cloud.

Introduction

Multi-channel audio in your source videos can be preserved in your Video Cloud video renditions. To do this, you need to have some special renditions and ingest profile(s) added to your account. You can have this done by contacting Brightcove Support.

Support will also help you create an ingest profile using these renditions, which you can then use when you are adding multi-channel audio videos.

Notes

  • VideoCloud supports mixing 5.1 surround sound down to Stereo -this will happen automatically if the source is a single multichannel 5.1 track, and stereo output is requested.
  • If the source has multiple mono audio tracks, the input_audio_channels option must be provided to specify a mapping of which track contains which channel. If you have multiple mono audio tracks, you will need to have custom ingest profiles created (as explained in the previous section) that include the input_audio_channels option with appropriate settings.
  • If you have source videos that use different audio track layouts, you may need multiple custom ingest profiles.
