    Overview

    Ooyala's CMS platform is named Backlot.

    Backlot documentation can be found at:

    http://api.ooyala.com/docs/v2

    https://docs.brightcove.com/apidocs-ooyala/backlot_api/

    https://help-ooyala.brightcove.com/video-platform/concepts/book_api.html


    Solution

    A translation proxy will be developed which handles Backlot API routes and paths; assimilates data from Brightcove Video Cloud APIs and responds with data structures that match the existing Backlot API (v2) format.

    In addition to the translation proxy, a service will be built to map the following constructs from Backlot to Video Cloud:

    Backlot construct
    Video Cloud construct
    Backlot construct
    Video Cloud construct
    PCODEPolicy Key/Account ID
    API KeyClient ID
    API SecretClient Secret
