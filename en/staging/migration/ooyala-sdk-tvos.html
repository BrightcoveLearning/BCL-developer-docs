Ooyala to Video Cloud Migration: Native SDK for tvOS

In this topic, you will learn how features supported by the Ooyala SDKs compare with the features supported by the Brightcove Native SDKs.

Overview

The Native SDKs allow you to deliver engaging video experiences which leverage the native player in mobile devices and connected TV.

Ooyala

Ooyala has two SDKs for the tvOS platform:

  • Ooyala SDK - This contains the core playback functionalities for the player. This is a direct port of the iOS core SDK, as the codebase is almost the same.
  • Ooyala Skin SDK - This is an open-source layer that manages the visual part of the player. Users can customize and add functionality to this layer.

Documentation: Ooyala Mobile SDK for tvOS

Brightcove

The Brightcove Native SDK for tvOS is contained within the SDK for iOS. The codebase contains the core functionality for the player and allows you to create custom control buttons.

Documentation: Brightcove Native SDK for tvOS
