Using the Pulse plugin with the Native SDK for Android

This is a temporary STAGING document for review only. For the official Brightcove documentation site, go to support.brightcove.com
In this topic, you will learn how to use the Pulse plugin with Brightcove's Native SDK for Android.

Introduction

Brightcove's Pulse plugin enables you to integrate Invidi's Pulse SDK with the Brightcove Native SDK for Android. Pulse is a Video Advertisement Platform. For campaigns and configuration details, see their User Guide.

Data

Support is defined in terms of the [major version].[minor version] of an OS (for example: Android 10.0). Customers can still use the Brightcove SDK for Android on unsupported versions, however we provide no guarantee of functionality.

Google Play Store

Google’s Play Store provides usage data for Android versions which have logged into the Play Store for a highlighted week. As of May 7th 2019, data shows that Android 5.0 and 5.1 account for 3.0% and 11.5% of total Android use respectively.

Current Play Store data can be found in the Android Distribution dashboard.
