Using the SSAI Plugin with the Native SDK for Android

This is a temporary STAGING document for review only. For the official Brightcove documentation site, go to support.brightcove.com

In this topic, you will learn how to use the Server-Side Ad Insertion (SSAI) plugin with the Brightcove Native SDK for Android. It also details how to migrate from using the OnceUX plugin.

Overview

Google continues to update the ExoPlayer media player framework with the latest technology solutions. In order to focus on cutting edge features, we must evolve alongside Google by providing support for only the relevant version of the ExoPlayer.

The ExoPlayer 1 release was last updated on March 17, 2017. If you haven't already done so, we recommend transitioning to the ExoPlayer2 framework.

For details about how to migrate, see the Migrating to ExoPlayer 2 Framework document.

Questions
