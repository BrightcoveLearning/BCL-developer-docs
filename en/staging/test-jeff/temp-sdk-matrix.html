Native SDKs Feature Support

This is a temporary STAGING document for review only. For the official Brightcove documentation site, go to support.brightcove.com

The table below lists the features supported by the Brightcove Native Player SDKs.

Overview

The Brightcove Native SDKs include Android, iOS and tvOS. They support the following frameworks:

  • Android - Libraries built on top of the Android ExoPlayer project (ExoPlayer 2 v2.7.0) [1]
  • iOS/tvOS - Libraries built on top of the Apple AVFoundation framework

Unless otherwise specified, a "Yes" in the table indicates that the feature is supported for all of the OS versions currently supported by the Native SDK. For a list of supported OS versions, see the following:

  • Native SDK for Android supported versions
  • Native SDK for iOS supported versions
  • Native SDK for tvOS supported versions
