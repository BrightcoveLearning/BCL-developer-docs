Doc Title

In this topic, you will learn about the bc() and getPlayer() methods, including the function of each and best practices of their use.

Introduction

A basic requirement when doing development with Brightcove Player is obtaining a reference to the player. The best practice for performing this has changed over the life of Brightcove Player 6.x. Here are best practices you should follow (details for each bullet follow in the document):

  • When using a version of the <video> tag on the page and you want to get a reference to the player do one the following:
    • v6.16.0+ use the player.getPlayer() method.
    • Before v6.16.0 use the videojs('playerID').ready() method.
  • When using the Advanced (in-page embed) player implementation, the best practice is:
    • v6.11.0+ use the <video-js> tag.
    • Before v6.11.0 use the <video> tag.
  • If you are manually instantiating the player on the page, use the bc() method.

getPlayer() versus ready()
