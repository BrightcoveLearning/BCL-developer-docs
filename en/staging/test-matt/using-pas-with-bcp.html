Introduction

Brightcove's Playback Authorization Service (PAS) offers an extra level of security when using Dynamic Delivery with DRM-protected or HTTP Live Streaming Encryption (HLSe) content.

With PAS, license requests will be authenticated using a signed JSON Web Token (JWT).

It is used when requesting the video license, once the video has been loaded to the player and the source has been selected.

With HLSe content, when the player changes rendition, due to improved connectivity or after the first segment is loaded, a new license request is made.

For more information about PAS, see the Overview: Playback Authorization Service document.

Brightcove Player implementation

There are a number of possibilities for implementing PAS with Brightcove Player. They are listed here, then detailed in later sections of this document.

Using a token in the catalog parameters

The first approach is to pass the token as a string as part of a catalog parameters object, using the property name bcovAuthToken. to any of the available methods. The get method is recommended because it returns a Promise.

This approach will work for both DRM as well as HLSe. The player will detect the type of source being loaded from the Playback API and provide the correct implementation for that source.

Note: This implementation requires Player version 6.33.0 or greater.

This sample implementation code uses the catalog.get() method to request the video while supplying the token (which would need to replace the <BCOV_AUTH_TOKEN>). This code uses a JavaScript Promise, if you would like to learn more about this concept see the Concepts: Using JavaScript Promises document.

<video-js id="myPlayerID"
  data-account="1507807800001"
  data-player="default"
  data-embed="default"
  controls
  data-application-id></video-js>
<script src="//players.brightcove.net/1507807800001/default_default/index.min.js"></script>

<script>
  (function() {
    var myPlayer = videojs.getPlayer('myPlayerID');

    myPlayer.catalog.get({
      type: 'video',
      id: '6015247091001',
      bcovAuthToken: '<BCOV_AUTH_TOKEN>'
    })
      .then(function(videoReturned){
        myPlayer.catalog.load(videoReturned);
      })
      .catch(function(err){
        console.log('err:', err);
      });
  })();
</script>
