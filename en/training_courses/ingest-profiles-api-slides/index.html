Using the Ingest Profiles API

The Ingest Profiles API allows you to manage ingest and transcode profiles for your account. This tutorial will walk you through these operations.

Getting Started

The Ingest Profiles API

The API allows you to:

  • Get the profiles for your account

  • Create new profiles

  • Update profiles

  • Delete profiles

  • Set or update the default profile for your account

The base URL

The base URL for the Ingest Profiles API is:

https://ingestion.api.brightcove.com/v1/
