Dimensions and Metrics

This document is an overview of the dimensions collected for the video_cloud domain, and the metrics exposed by the API. More information for a particular dimension can be found in the documentation for that dimension.

Dimensions: video_cloud

The video_cloud domain collects data for the following dimensions:

Dimension Explanation
account Video Cloud account id.
player Video Cloud player id (belonging to the account).
video Video Cloud video id (belonging to the account).
referrer_domain Top private domain of the referrer when events are collected
