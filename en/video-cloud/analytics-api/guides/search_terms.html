search_terms Dimension

The search_terms dimension is the keywords used if the traffic source that resulted in events being collected for activity metrics was a search engine, and the keywords are detectable. The search_terms dimension only includes metrics from player instrumentation events.

search_terms in the API

Search phrases are recorded largely unchanged, except:

  1. Case-sensitivity is ignored: events with searches for "brightcove", "Brightcove", and "BRIGHTCOVE" all count toward "brightcove".
  2. Leading and trailing whitespace is trimmed.
  3. Internal spaces are collapsed to a single space.
  4. Any search phrase with more than 100 characters after applying #2 and #3 is counted as "...".

Report Data

Report data can be broken down by search_terms. In addition, report data broken down by search_terms can be filtered for a specific video or player using the where parameter.

Example

https://analytics.api.brightcove.com/v1.data?accounts=12345,654321&dimensions=search_terms&limit=3&sort=video_view
