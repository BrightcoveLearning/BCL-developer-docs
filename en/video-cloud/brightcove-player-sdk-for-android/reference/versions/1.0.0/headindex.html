Brightcove Android SDK Reference

The Brightcove Android SDK enables you to develop your own Android native application to deliver your Brightcove videos to Android platforms. This document provides reference information about the two libraries that make up the Brightcove Android SDK:

  • Brightcove Media API Library (version 1.0.0)
  • Brightcove Android Media Player Library (version 1.0.0)
