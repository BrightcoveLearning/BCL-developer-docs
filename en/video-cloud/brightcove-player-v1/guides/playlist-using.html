Using Playlists

Note: Playlists in Brightcove Player are in beta development.

A playlist is a collection of videos that are grouped together in a particular order for playback in Brightcove Player. The following shows a Brightcove Player with an associated playlist. You can click on a video in the playlist to load that video into the player.

    To use playlists, complete the follow tasks:

    1. Create a playlist either in Video Cloud Studio or manually using JavaScript
    2. Enable your player to use playlists
    3. Use playlists with the iframe player implementation
    4. Use playlists with the in-page embed implementation
    5. Use playlists with the Playlist API

    Enable player

    At some time in the near future you will be able use Video CloudPerform Studio to enable your player to use playlists. Until that time, you need to add a "playlist": true property to your player. You need to EITHER use the form below to perform the task, OR use a curl statement.

    Enable player using a form

    To use the form you need the following information:

    • Client ID and client secret - An instructional video and instructions on how to obtain these credentials can be found in the Managing API Credentials document. When selecting permissions select Players > Read/Write.
    • Account ID - Obtained from the Home page of Video CloudPerform Studio.
    • Player ID - Obtained from the Players page of Video CloudPerform Studio.

    The form will securely submit the information to a proxy, which will then make the API call to update your player to use playlists by setting the playlist value. On successful update, the player will then be published using another API call.

    Enable player using curl

    Note: You need to use the following curl statement ONLY IF you choose not to use the form to playlist enable your player.

    If you would rather not use the form, and have the proper skill set, you can patch your player using a curl statement. To use the curl statement shown below, be sure you have checked the Enable Basic Authentication button in Video CloudPerform Studio administration settings:

    The actual curl statement is provided here:

    curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL \
  --request PATCH \
  --data '{
      "playlist": true
  }' \
  https://players.api.brightcove.com/v1/accounts/YOUR_ACCOUNT_ID/players/YOUR_PLAYER_ID/configuration

    Be sure to publish your player after making the change.

    Note: Using curl is detailed in the Quick Start to Player Management document.

    iframe player

    Once you have a playlist to use, and have enabled your player to use a playlist, you are ready to actually utilize the playlist with the player. As usual you have the iframe and in-page embed player implementations available. In this section, instructions for using the playlist with the iframe implementation are shown. When using the iframe, the playlist will appear on the right side of the player, as shown in this screenshot (NOT a functional player and playlist):

    Note: If you need to position the playlist in a different location other than on the right side of the player, you will need to use the in-page embed implementation, detailed below.

    To use a playlist with the iframe implementation, you append one of the following query strings to the src attribute of the iframe. If using an ID:

    <iframe src='//players.brightcove.net/.../index.html?playlistId=1754200320001'

    or if you are using a reference ID:

    <iframe src='//players.brightcove.net/.../index.html?playlistId=ref:myrefid001'

    If you added one of the query strings, then browsed the page with your iframe player implementation, chances are you would not see the playlist.     In-page embed player

    Once you have a playlist to use, and have enabled your player to use a playlist, you are ready to actually utilize the playlist with the player. As usual you have the iframe and in-page embed player implementations available. In this section, instructions for using the playlist with the in-page embed implementation are shown.

    To use the playlist in the in-page embed implementation you need to do the following:

    • Add one of the following attributes to the <video> tag (the first is using a playlist ID, the second is using a reference ID):
