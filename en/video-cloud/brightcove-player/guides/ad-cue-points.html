Display Ads using Ad Cue Points

In this document, you will learn how to use the IMA3 plugin and ad cue points to display ads.

Implementation overview

To play ads based on ad cue points in the video, you will need to perform the following:

  • In Studio's MEDIA module, create the ad cue points in the video at your desired time(s).
  • Create a VAST tag that will serve the desired ads to the video when a cue point triggers a request.
  • Be sure the IMA3 plugin is loaded in the player. In this doc, you will see two ways to perform this task.
  • Configure the IMA3 plugin to use ad cue points.

Create ad cue points

You can use Studio's MEDIA module to ad cue points to a video. Click on a video, then perform the following tasks:

  1. Just below the video, click the Ad Cue Point button.
  2. Enter the time you want the cue point to be dispatched.
  3. Set the Type to ad.
  4. Give the cue point a name.

You may add multiple cue points.

Create/obtain a VAST tag

When configuring the IMA3 plugin you will need to provide a URL to a location of a VAST tag. The VAST tag serves the ad for the player to play. In this example, a static VAST tag, which is actually a URL to a VAST XML document, is used. You may wish to use an ad server to randomly choose from multiple ads to be viewed. Regardless, the ad cue point will need to point to a valid VAST tag.
