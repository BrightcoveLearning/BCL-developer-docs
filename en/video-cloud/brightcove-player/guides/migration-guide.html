Smart Player to Brightcove Player Migration Guide

This guide will help you transistion from working with Brightcove's Smart Player to using the new Brightcove Player. You can learn more about this new player in the What is the New Brightcove Player? document.

Plugins

Smart Player

When creating plugins for the Smart Player, you have to do both:

  • Write a JavaScript plugin for when the player is in HTML5 mode
  • Write an equivalent SWF version of the plugin for when the player is in Flash mode

Brightcove Player

When creating plugins for the Brightcove Player, you only have to:

  • Write a JavaScript version of the plugin

Methods

This table shows the common methods used in the Smart Player and their equivalent counterparts in the Brightcove Player:

  • For details about the Smart Player methods, see the Smart Player API Reference document.
  • For details about the Brightcove Player methods, see the Brightcove Player API document.
Smart Player Brightcove Player Description
templateLoadHandler() Loads the data for the player and the API
templateReadyHandler() ready() Returns when the player is loaded and ready
getExperience(id) videojs(id) Get a reference to the player
getModule(APIModules. VIDEO_PLAYER) Get a reference to the player's API modules
play() play() Start media playback
pause() pause() Pause video playback
addEventListener( event, handler, priority) on(first, second, third)[1] Smart Player: Add an event handler for the specified event.
BC Player: Add an event listener for a component.
removeEventListener( event, handler) off([first], [second], [third])[1] Smart Player: Remove the event handler for an event.
BC Player: Remove an event listener from a component.
getVideoDuration() duration() Get the video length in seconds
getCurrentVideo( callback with video DTO) Get the rendition DTO for the currently playing video
catalog.getVideo(videoId, callback) Get video object by Id
loadVideoByID(id) catalog.load(video) Smart Player: Load a video for a given Id
BC Player: Load a video object into the player. The video object is returned from the catalog.getVideo() method.
seek(seconds) currentTime(seconds) Seek to a specified time position in the video
getVideoPosition(format, callback) currentTime() Get the current time in seconds

Notes

  • [1] first: String|vjs.Component event type or component; second: Function|String event handler or component; third: Function event handler

Events

This table shows the common events used in the Smart Player and their equivalent counterparts in the Brightcove Player:

  • For details about the Smart Player events, see the Smart Player API Reference document.
  • For details about the Brightcove Player events, see the Brightcove Player API document.
Smart Player Brightcove Player Description
templateLoad All data for the player and the API has been received by the browser
templateReady The player is instantiated and is ready to interact with the API
loadstart Fired when the user agent begins looking for media data
loadedmetadata Fired when the player has initial duration and dimension information
loadeddata Fired when the player has downloaded data at the current playback position
MediaEvent.BEGIN myPlayer.one('play'); Fired the first time a video is played
MediaEvent.PLAY play Media has begun or resumed playback
MediaEvent.STOP pause Media has been paused or stopped
progress Fired while the user agent is downloading media data
MediaEvent.PROGRESS timeupdate Fired when the current playback position has changed
MediaEvent.COMPLETE ended Fired when the end of the media resource is reached
MediaEvent.CHANGE Fired when the media has changed in the player
seeking Fired when the media is being seeked to a new position
seeked Fired when the media has been seeked to a new position
MediaEvent.ERROR Signifies there was an error loading media
durationchange Fired when the duration of the media resource is first known or changed
fullscreenchange Fired when the player switches in or out of fullscreen mode
resize Fired when the width and/or height of the component changes
volumechange Fired when the volume changes
waiting Fired whenever the media begins waiting
