Player Application ID for Analytics

In this topic you will learn how to use an application ID for a player. This functionality allows you to re-use a single player, but differentiate analytics on a per-site or per-application basis.

Understanding the application ID

You may have multiple websites, but use the same Brightcove Player on the different sites. But, you want player analytics on a per-website basis. Previously, this could only be achieved by creating a player for each website, even if the players were otherwise identical. Using an application ID will allow you to create a single player, but differentiate it at runtime so you get per-website analytics.

The value of the application ID is arbitrary and left to your use-case. The only place the ID surfaces is in analytics, where the application ID is appended to the player name. For example, if there is a player named My Player and it has the application IDs Hello and Goodbye provided in context, the analytics module will display results for the contexts as My Player - Hello and My Player - Goodbye.

Setting the application ID

You can set the application ID in either the Standard (iframe) or Advanced (in-page) player implementations.

For the Standard (iframe) implementation, you use a query parameter named applicationId to define the application ID, as shown here:

<iframe src="//players.brightcove.net/1507807800001/BkTGbgSq_default/
index.html?videoId=5079788144001&applicationId=MyAppID"
  allowfullscreen
  webkitallowfullscreen
  mozallowfullscreen></iframe>
<iframe src="//players.brightcove.net/1507807800001/BkTGbgSq_default/
index.html?applicationId=MyAppID"
  allowfullscreen
  webkitallowfullscreen
  mozallowfullscreen></iframe>

For the Advanced (in-page) implementation, you use an attribute in the <video> tag named data-application-id to define the application ID, as shown here:

<video data-video-id="5079788144001"
  data-application-id="MyAppID"
  data-account="1507807800001"
  data-player="BkTGbgSq"
  data-embed="default"
  class="video-js"
  controls></video>
<script src="//players.brightcove.net/1507807800001/BkTGbgSq_default/index.min.js"></script>
<video data-application-id="MyAppID"
data-account="1507807800001"
data-player="BkTGbgSq"
data-embed="default"
class="video-js"
controls></video>
<script src="//players.brightcove.net/1507807800001/BkTGbgSq_default/index.min.js"></script>
