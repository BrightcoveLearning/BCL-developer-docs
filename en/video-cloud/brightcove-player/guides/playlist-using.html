Using Playlists

A playlist is a collection of videos that are grouped together in a particular order for playback in Brightcove Player. The following shows a Brightcove Player with an associated playlist.

    To use playlists, complete the follow tasks:

    1. Create a playlist either in Video Cloud Studio or manually using JavaScript
    2. Enable your player to use playlists
    3. Use playlists with the iframe player implementation
    4. Use playlists with the in-page embed implementation
    5. Use playlists with the Playlist API

    Enable player

    At some point in the near future you will be able to playlist enable your player. Until that work is done, you need to add a "playlist": true property to your player. You can use the form below to perform the task, or use curl.

    Enable by filling in a form

    To use the form you need the following information:

    • Client ID and client secret - An instructional video and instructions on how to obtain these credentials can be found in the Managing API Credentials document.
    • Account ID - Obtained from the Home page of Video Cloud Studio.
    • Player ID - Obtained from the Players page of Video Cloud Studio.

    The form will securely submit the information to a proxy, which will then make the API call to update your player. On successful update, the player will then be published.

    Enable by using curl

    If you would rather not use the form, and have the proper skillset, you can patch your player using this curl statement:

    curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
      "playlist": true
    }' \
  https://players.api.brightcove.com/v1/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration

    Be sure to publish your player after making the change.

    Note: Using curl is detailed in the Quick Start to Player Management document.

    Playlist API

    The Playlist API gives you programmatic control over the usage of playlists, for instance next(), previous() and autoadvance() methods. The Playlist API Guide document provides full details.
