Vocabulary Review:

  • When a method retrieves a value it is called a getter.
  • When a method assigns a value it is called a setter.
  • When defining a method the placeholder for the value that is passed to the method when called is a parameter, e.g. function myMethod(parameter).
  • When calling a method the value in the parentheses is called an argument, e.g. myMethod(argument).

NOTE: It is not unusual for one method to act both as a getter and setter. When using the method if an argument is passed the method acts as a setter, with no argument the method acts as a getter.

playlist( ) method

The playlist(newPlayList) method allows you to either retrieve the playlist currently playing in the player, or assign a playlist to the player to play. If called without an argument the method performs the retrieval and acts as a getter, with an argument is assigns the value and acts as a setter.

Parameters:
  • newPlaylist: An array of video objects. The objects can either be manually created or a Video Cloud playlist.
