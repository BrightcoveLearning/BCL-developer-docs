Smart Player Migration Guide

This guide will help you transistion from working with Brightcove's Smart Player to using the new Brightcove Player. You can learn more about this new player in the What is the New Brightcove Player? document.

Plugins

Smart Player

When creating plugins for the Smart Player, you have to write both a JavaScript plugin for when the player is in HTML5 mode and an equivalent SWF version of the plugin for when the player is in Flash mode.

Brightcove Player

When creating plugins for the Brightcove Player, you only have to write a JavaScript version of the plugin.

Methods

This table shows the common methods used in the Smart Player and their equivalent counterparts in the Brightcove Player:

Smart Player Brightcove Player Description
templateLoadHandler() Loads the data for the player and the API
templateReadyHandler() ready() Returns when the player is loaded and ready
brightcove.api.getExperience(id) videojs(id) Get a reference to the player
getModule( APIModules.VIDEO_PLAYER) Get a reference to the player's API modules
play() play() Start media playback
pause() pause() Pause video playback

Events

This table shows the common events used in the Smart Player and their equivalent counterparts in the Brightcove Player:

Smart Player Brightcove Player Description
templateLoad All data for the player and the API has been received by the browser
templateReady ready The player is instantiated and is ready to interact with the API
