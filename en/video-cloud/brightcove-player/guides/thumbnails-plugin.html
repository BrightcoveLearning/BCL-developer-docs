Display Thumbnail Previews Plugin

The thumbnail previews plugin allows for the timeline to display a preview image at defined points for your video. The user can mouse over the timeline and images defined by you will be displayed for specified time ranges.

Collect images

At this time you must collect the images yourself. Preview thumbnails can be generated as part of the Zencoder encoding process. Also, tools exist like SnapMotion to automate this process.

No matter how you generate the images, you need to decide the number of images you wish to use and choose an appropriate size.

Implement plugin

The plugin's JavaScript and CSS are already written for you. To use the plugin you need to configure a player with the proper configuration information. In the data property from the curl statement shown below you see an example of including the plugin with a player. You see the three main properties for which you need to supply values. They are:

  • scripts: This is the JavaScript supplied to you for the plugin. This will not change for different plugin implementations.
  • stylesheets: This is the CSS supplied to you for the plugin. This will not change for different plugin implementations.
  • plugins: This property has two values. The first is name with the associate value "thumbnails". The name thumbnails is defined in the supplied JavaScript for the plugin, and this will not change for different plugin implementations. The second property is options, which is discussed in detail below.

options property

For plugins in general, the options property is used to pass data to the plugin for utilization at initialization time. In this case, you will use this property to build an object that contains key-value pairs to define times when a particular thumbnail preview image should begin to be displayed, and the URL to the thumbnail preview image. The options tell the player how to behave when the mouse hovers over the timeline. The example JSON shown below that is the value for the options property causes the plugin to behave as described in the following bullet points:

  • The Tiger.jpg image is be displayed from video start up to 5 seconds
  • The Water.jpg image is displayed from 5 seconds up to 10 seconds
  • The BirdsOfAFeather.jpg image is displayed from 10 seconds to the end of the video
