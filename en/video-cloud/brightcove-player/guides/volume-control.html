Volume Control

In this topic, you will learn how to use two methods, muted() and volume(), to programmatically control audio for Brightcove Player. Of course you can manually control audio using the volume button in the player controls.

Note: The examples below that use the volume() and muted() methods are executed when the player is ready. You need not wait for the metatdataloaded event, for example, since the volume is a player setting and independent of the video being played.

Player example

volume() method

The volume() method acts as both a getter and a setter depending upon if an argument is passed. If the method is used as a setter, the argument is a decimal number between 0 (muted) and 1.0 (full volume). The argument is a percentage represented as a decimal number. If the method is used as a getter, it returns the decimal value of the volume.

Below is a code example programmatically using the volume() method to set the volume to 60%.

videojs('myPlayerID').ready(function(){
  var myPlayer = this;
  myPlayer.volume(.6);
});

This code toggles volume between 30% and 100%:

function toggleVolume(){
  if (myPlayer.volume() > .3) {
    myPlayer.volume(.3);
  } else {
    myPlayer.volume(1);
  }
}

muted() method

The muted() method acts as both a getter and setter depending upon if an argument is passed. If the method is used as a setter, the argument is a boolean value, true to mute and false to unmute. If the method is used as a getter, the current boolean value is returned.

Below is a code example programmatically using the muted() method to mute player audio.

  videojs('myPlayerID').ready(function(){
    var myPlayer = this;
    myPlayer.muted(true);
  });
Note: There is a subtle difference between volume(0) and muted(true). The difference is that when muted is toggled the volume will be remain at whatever the percentage was before muted() was used. Of course, if volume is set to 0, to resume audio a percentage will have to be set again.

This code toggles the volume on and off:

function toggleMute(){
  if (myPlayer.muted()){
    myPlayer.muted(false);
  } else {
    myPlayer.muted(true);
  }
}

volumechange event

If you wish to react to a change in volume you can use the volumechange event. You can set the event listener as follows:

    myPlayer.on('volumechange', function( evt ){
      console.log('event: ', evt);
    })
