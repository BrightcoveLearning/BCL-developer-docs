Player Feature Support by Browser

Generally speaking, for best performance and feature support use the latest version of your OS and browser. Brightcove Player continually grows and improves and sometimes this can only be done using the most advanced features from the browser and OS on which the player is running.

Testing

You can test your specific video and features in Brightcove Player using the Player Reference. Although the Player Reference is quite easy to understand and use, the Using the Player Reference document has both a quick video and written instructions on the page's use.

Browser support

Brightcove Player is supported by the LATEST VERSIONS of following browsers.

Browser
Chrome on Windows, Mac OS and Android
Safari on Mac OS
Safari on iOS
...
Vendors of the browsers listed above release updates that usually maintain backward compatibility. However, the release of newer versions may precede Brightcove's ability to test and verify full compatibility.

Non-functioning browsers

The following are know versions of browsers were issues will arise, and should be avoided. This does NOT imply that all other browser versions are supported.

  • Internet Explorer 9 and earlier (IE9 and earlier are no longer supported by Microsoft and make up a tiny fraction of Brightcove's traffic, and are therefore no longer actively supported. Brightcove will consider fixing issues related to these browsers on a case-by-case basis.)
  • Safari version 5 and earlier on Mac OS
  • Safari version 6 and earlier on iOS
