Managing Playlists Using the CMS API

This topic describes how to create, update, or delete playlists using the CMS API.

Introduction

The Media API contains a set of write methods you can use to create, update, or delete playlists:

  • Create a playlist
  • Update a playlist
  • Delete a playlist
These tasks can also be performed in the Studio Media module.

Authentication

Requests to the CMS API require an authorization header which contain your access tokens. For details on how to obtain client credentials and use them to retrieve access tokens, see the Brightcove OAuth Overview.

GET videos

Before we get into creating playlists, let's see how we can retrieve data for all of the playlists in your Video Cloud account.

https://cms.api.brightcove.com/v1/accounts/1507807800001/playlists
