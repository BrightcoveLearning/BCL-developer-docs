Standard Ingest Profiles

When creating a Dynamic Ingest request using the DI API you must supply profile which determine the number and type of renditions created by the ingest. The standard profiles from which you can choose are:

  • audio-only
  • balanced-high-definition
  • balanced-standard-definition
  • Express Standard
  • high-bandwidth-devices
  • low-bandwidth-devices
  • mp4-only
  • screencast
  • single-rendition

Here are details of each of the profiles:

Name Min. Tier # HD Renditions # SD Renditions Comment Formats
audio-only VOD 0 0 Audio tracks to cover all devices MP4, HLS
screencast VOD 6 4 Larger frame size with lower bitrates MP4, HLS
single-rendition VOD 0 1 A single good quality MP4 MP4
