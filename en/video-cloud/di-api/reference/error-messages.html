Dynamic Ingest Error Message Reference

This is a reference to error messages returned by the Dynamic Ingest API. Also see the error message reference for the CMS API, which is used to create the video object before the video is ingested.

Error messages

The follow error messages can be returned by the Dynamic Ingest API

Name Message Cause
NOT_SUBMITTED Unable to submit job, please try again later. System issue - unable to create the job in the ingest system
INTERNAL_ERROR Internal error, please try again later. Issue with API
NO_SUCH_VIDEO Unable to find the referenced video. Bad URL for the video source file.
UNAUTHORIZED Unable to authorize request. Missing or bad OAuth credentials.
BAD_ACCOUNT Account ID was missing or invalid. Invalid account ID specified.
NO_SOURCE Unable to find a source to use. Unable to find a usable video file at the address specified.
PROFILE Unable to find profile by name. Could not find the specified ingest profile in the account
CDN_CREDENTIALS Unable to fetch CDN credentials Could not get the credentials for the account CDN
BAD_CALLBACKS Callbacks were not in expected format Callback URLs were not specified correctly.
