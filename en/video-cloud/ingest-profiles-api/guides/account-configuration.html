Manage Configurations

Now that we have looked at the operations on profiles, we will turn to operations on the profile configuration for a Video Cloud account. These operations allow you manage configurations that set the default profile for the account and also limit the number of profiles and renditions per profile. An account configuration looks like this:

{
             "account_id": 123456789,
             "default_profile_id":"Protected",
             "max_profile_count":10,
             "max_rendition_count":30
          }

List configurations

First we will see how you get a list of all ingest profiles configurations for the account.

Request

This will be a GET request, and the endpoint to get the configurations is:

/accounts/{accountId}/configuration

Create a configuration

Now let's see how you create a new ingest profiles configuration for the account.

Request

This will be a POST request, and the endpoint to create configuration is:

/accounts/{accountId}/configuration

Get a configuration

Next we will see how you get a particular ingest profiles configuration for the account.

Request

This will be a GET request, and the endpoint to get a configuration is:

/accounts/{accountId}/configuration/{configuration-id}

Update configuration

Next we will see how you update a ingest profiles configuration for the account.

Request

This will be a PUT request, and the endpoint to update a configuration is:

/accounts/{accountId}/configuration/{configuration-id}

Delete configuration

Next we will see how you delete a ingest profiles configuration for the account — note that deletes can not be undone!

Request

This will be a DELETE request, and the endpoint to delete a configuration is:

/accounts/{accountId}/configuration/{configuration-id}
