Standard Ingest Profiles

In this topic, you will learn about the properties of the standard ingest profiles for your Video Cloud account. When creating a Dynamic Ingest request using the DI API or uploading videos using the Upload module, you must supply a profile which determines the number and type of renditions created by the ingest.

Notes:
  • This page is built using live data from the API; it may take a minute to fully load.
  • Standard profiles may not be edited - if you want a modified version, you will need to create a custom profile.
