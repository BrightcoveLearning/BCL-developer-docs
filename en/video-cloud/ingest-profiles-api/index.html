Brightcove Video Cloud Ingest Profiles API

Note: the Ingest Profiles API is currently in Limited Availability (LA) mode. Participation in the LA program requires approval by the product manager. If you are interested in participating, contact your account manager.

The Ingest Profiles API is a REST-based API that allows you to create, retrieve, update, and remove ingest profiles for your Video Cloud account. Since data is retrieved using HTTP requests, you can use any programming language to make requests and use the data to build solutions for managing your ingest profiles.

If you are new to this API, you will probably want to start with the Getting Started guide. Next, look at the API Overview.Once you master the basics, find more details on the endpoints and operations in the API Reference.
