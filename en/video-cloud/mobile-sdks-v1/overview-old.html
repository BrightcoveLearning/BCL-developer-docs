Brightcove Mobile SDK Overview

The Brightcove player is an enhanced version of the native media player for Android and iOS.

Brightcove player SDK advantages

Using the Brightcove player SDK for Android and iOS, you get many benefits:

  • Better performance
  • Videos are quick to load
  • Custom overlays and click-through ads

You can also easily integrate the following functionality:

  • Analytics
  • Digital rights management (DRM)
  • Advertising
  • Access to your media library data

Mobile web limitations

While you can use a web for mobile implementaion instead of using the SDKs, there are some limitations with this approach:

  • Requires a lot of overhead
  • Videos will take some time to load
  • No overlays or click-through ads

Android Architecture

The Brightcove SDK for Android is built on top of existing Android tools, simplifying video playback and the use of your Video Cloud videos and playlists.

The BrightcoveVideoView object replaces the native Android VideoView object.

Analytics

If you use the Brightcove player and Catalog included in the SDK to retrieve and play videos, analytics data will be automatically collected and will appear in your Video Cloud Analytics.

If you override the BrightcoveVideoView class or do not use the Brightcove player and catalog, you need to send your Publisher ID to Video Cloud Analytics.

For details about sending data to Video Cloud Analytics, see the Using Brightcove Analytics guide.

iOS Architecture

The Brightcove SDK for iOS is built on top of existing iOS tools, simplifying video playback and the use of your Video Cloud videos and playlists.

The BCOVPlayerSDKManager object is the entry point for the Brightcove player.

Analytics

If you use the Brightcove player and Catalog included in the SDK to retrieve and play videos, analytics data will be automatically collected and will appear in your Video Cloud Analytics.

If you override the BCOVVideo class or do not use the Brightcove player and catalog, you need to send your Publisher ID to Video Cloud Analytics.

For details about sending data to Video Cloud Analytics, see the Using Brightcove Analytics guide.
