Video Cloud App SDK for iOS Reference

Note: this legacy documentation is for a deprecated version ok the SDK for iOS. You should use the new Brightcove The Brightcove Video Cloud Player SDK for iOS instead.

The Video Cloud App SDK for iOS enables you to develop your own Apple iOS native application to deliver your Video Cloud videos to the iPhone, iPad, and iPod Touch platforms. This document provides reference information about the three libraries that make up the Video Cloud App SDK for iOS:

  • Video Cloud Media API Library (version 2.0.0)
  • Video Cloud iPad iPhone Kit Library (version 2.0.0)
  • Video Cloud iPhone Kit Library (version 1.2.0)(deprecated)

Read more in Getting Started with the Video Cloud App SDK for iOS help topic, which also includes links to other resources.
