Brightcove Player SDK for tvOS: Previous SDK information

This topic shows the previous SDK and plugin versions for the Native Player SDK for tvOS.

The following links for the previous version of the Brightcove Native Player SDK for tvOS is for reference only. It is recommended to use the latest version of the SDK along with the plugin integrations.

The Brightcove Player SDK for tvOS is contained within the Player SDK for iOS. The following links are for the previous version of the Brightcove Native Player SDK for iOS.
