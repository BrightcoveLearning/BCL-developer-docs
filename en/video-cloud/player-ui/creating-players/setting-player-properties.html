Setting Player Properties

This topic provides an overview on how to modify player settings in the Players module.

Setting player properties

To configure or edit player settings, open the Players module and click on a player link.

The following player properties can be configured:

  • Player Information
  • Settings
  • Styling
  • Plugins
  • Viral Promotion
  • Video Content

Player Information

The Player Information properties allow you to customize the player name and short description. To edit the Player Information properties:

  1. Click the Edit button.
  2. Enter a name and short description for the player.

  3. Click Save to save the changes.

Note: The Player ID assigned to the player cannot be changed.

Settings

The settings properties are used to configure video auto-start, fullscreen options, and the player dimensions. For more information, see Configuring Player Settings.

Styling

The styling properties are used to configure the icon, progress bar and background colors. For more information, see Styling Players.

Plugins

Player plugins use a combination of HTML, JavaScript and/or CSS to customize the behavior of the player. Plugins can be developed to:

  • Modify the default behavior of a player
  • Add functionality to a player
  • Customize player appearance

For more information, see Configuring Player Plugins.

Viral Promotion

The viral promotion properties are used to configure the viral sharing options for the player. For more information, see Configuring the Viral Promotion Properties for a Player.

Video Content

The video content properties are used to configure the poster image and video to display inside the player. For more information, see Configuring the Video Content Properties for a Player.
