Policy API Overview

The Policy API is used to create or get policy keys that can be used to access the Playback API.

Use cases

The Brightcove Player and the Brightcove SDK players include catalog methods identical to methods of the Policy API, and for the most part you can simply use those if you are just retrieving videos or playlists at runtime to play.

However, the catalog methods are restricted to retrieving 10 items or less per second. There might be occasions when you need to retrieve more items - for example, you might want to use the Analytics API to find your 20 most popular videos over the past week and display those as a playlist.

In addition, there may be occasions when you want to retrieve videos or playlists to display information about them, without having a player on the screen, as the landing page for a video portal, for example.

The Policy API is useful in these instances.

General Information

Base URL

The base URL for the Policy API is:

https://edge.api.brightcove.com

Account path

In all cases, requests will be made for a specific Video Cloud Account. So, you will always add the term accounts followed by your account id to the base URL:

https://edge.api.brightcove.com/accounts/{account_id}

Authentication

Requests are authenticated by a policy key, which can be passed in one of three ways:

  1. In an argument to an Accept header: 
    Accept: application/json;pk={policy_key}
  2. In an Authorization header using the realm keyword BCOV-Policy: 
    Authorization: BCOV-Policy {policy_key}
  3. In a BCOV-Policy header: 
    BCOV-Policy: {policy_key}

Methods

The API supports the following GET requests only, of the following types:

  • get video by id
  • get video by reference id
  • get playlist by id
  • get playlist by reference id

Video requests

A single video object can be retrieved using either the video id or reference id.

Get video by id

Endpoint

https://edge.api.brightcove.com/accounts/{account_id}/videos/{video_id}

https://edge.api.brightcove.com/accounts/57838016001/videos/38467382999
