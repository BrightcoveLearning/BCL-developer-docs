Uploading Videos Using the Upload Module

This topic provides an overview of the Upload module which is used to used to upload videos to a Video Cloud account.

Note: The New Video Cloud Studio is currently in beta and is not supported by Brightcove Support. If you would like access to the beta, please contact your Brightcove Account Manager.

Using the Upload module

To access the Upload module, login to Video Cloud Studio and click Upload in the navigation header.

The Upload module can also be accessed at: //studio.brightcove.com/products/videocloud-pre-release/upload

This topic contains the following sections:

  • Uploading Videos
  • Setting video properties

Uploading Videos

The Upload module uses a dynamic ingest process to retrieve and transcode video content. When you select a file to upload, Video Cloud uses a pull-based process to pull the content from a storage location where it then uses the Zencoder transcoding service to transcode the content. The video renditions that are generated are based upon the ingest profile that is configured for your account NOT the transcode settings that are configured as part of your Video Cloud account settings.

Note: Currently, all accounts are configured with the balanced-high-definition profile. There is no option to select another profile in the Upload module but that functionality will be added in a future update.

Files can be selected using these options:

  • Drag and Drop - Drag files from the file system to the outlined area.
  • File selection dialog - Click the Select Files link to open a file selection dialog to browse and select the files to upload.
  • Remote URL - Click the Enter URL link to specify the remote file location to upload from. File locations may include a public DropBox folder or Amazon S3 storage location.

Each file will display the file status:

  • Waiting - The file is waiting to be uploaded
  • Uploading - The file is being uploaded
  • Processing - The file is being transcoded
  • Processing Complete - The file transcoding is complete
  • Cancelled - The file upload was cancelled

Note: To view the files in your account and to publish videos, use the Media module.

Setting video properties

To edit video properties, click the Edit button.

The following properties can be edited:

  • Video Title - Title for the video
  • Description - Short description for the video
  • Tags - Tags associated with the video
  • Reference ID - Reference ID for the video

Click Save to save the changes. To learn more about video properties, see Editing Basic Properties.
