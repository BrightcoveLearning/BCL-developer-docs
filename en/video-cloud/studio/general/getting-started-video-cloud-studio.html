Getting Started with the New Video Cloud Studio

This topic provides an overview of the new Video Cloud Studio interface and modules. Video Cloud Studio allows you to manage your video publishing workflow across desktop, tablet and mobile devices.

To access Video Cloud Studio, navigate to: https://studio.brightcove.com/products/videocloud-pre-release/home. Sign in with your Email address and password. The homepage provides the ability to:

  • View system notifications
  • Manage Video Cloud settings
  • Select an account and view the user profile
  • Access Video Cloud modules
  • View the Video Cloud dashboard

Viewing system notifications

The Video Cloud Studio header will display a bell icon when there are system notifications available. Clicking the icon will display the list of system notifications.

Accessing modules

The Video Cloud Studio header provides access to the Studio modules.

Note that the modules that display are based upon your Video Cloud subscription and user profile.

  • Upload - Upload videos to your Video Cloud account
  • Media - Set video properties, organize videos using folders and publish videos
  • Players - Create and style players and generate publishing code for players
  • Analytics - View analytics on video and player performance
  • Live - Create and stream live events
  • Gallery - Create video portals for your video content

Selecting an account

If you have access to multiple Video Cloud accounts, the Video Cloud account selector can be used to select the Video Cloud account you want to access. To switch accounts, click on your name - account in the top right corner of Video Cloud Studio. The current account is indicated with a dot preceding the account name.

Click the My Profile link to update your user information, password and email address. For more information, see Managing Your Profile.

Managing settings

The Video Cloud Studio settings menu can be accessed from the settings icon . The Video Cloud settings menu provides access to:

  • Manage Users - Manage the users who are authorized to use Video Cloud Studio; Note: you must be an account administrator to access this option
  • Account Information - View information about the current Video Cloud account
  • API Authentication - Create and manage API credentials
  • Player Service Settings - Provides the option of using basic user name and password authentication for access to the Player Management and Delivery System APIs

Note: You must be an account administrator to have access to the account settings.

Viewing the dashboard

The Video Cloud Studio homepage displays the Video Cloud dashboard. The dashboard contains a series of "cards" which provide high level metrics on the videos in your account. Cards in the dashboard can be rearranged by clicking and dragging the arrow ()in the top right corner of the card. The following cards are available:

  • Analytics - Displays number of video views per day for the time period selected
  • Summary - Displays total videos and players in the account and recently created searches
  • Device Type - Displays type of devices used to watch videos during the last 30 days
  • Product Updates - Displays Video Cloud product update information
  • Top Trending Videos - Displays top three videos with the most views for the last 30 days
  • Geo Map - Displays viewer locations for the last 30 days

Localization

The Video Cloud Studio language can be changed by changing the language in your profile. Currently, English, Spanish and Japanese are supported. For more information, see Managing Your Profile.
