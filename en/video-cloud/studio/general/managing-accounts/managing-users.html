Managing Users

In this topic you will learn how to manage the users that have access to Video Cloud Studio.

You can use Video Cloud Studio to manage the users who are authorized to use the Studio. Note that to access the Users page, you must be a Video Cloud Studio user with the Administrator role.

To access the Users page:

  1. Login to Video Cloud Studio.
  2. On the home page, click the gear icon in the page header and then click Manage Users.

The Manage Users page will display.

The Manage Users page can be used to:

  • Create new users
  • Edit existing users
  • Delete a user

Creating new users

To create a new user:

  1. Enter the Email, First Name and Last Name of the user.
  2. Assign a Role to the user:
    • Administrator - User has access to account settings
    • Standard - User does not have access to account settings
  3. Select the modules the user has access to.

    Note: Currently only the Players module is available.

  4. Click Add a User.

The user will receive an email message stating that they have been added to the account.

Editing existing users

To edit an existing user:

  1. Click on the user to edit. The user details will appear at the top of the page.
  2. Currently only the Role can be changed.

    Note: The user that you logged in as cannot be edited.

  3. Click Save to save the changes.

Deleting a user

To delete a user:

  1. Click the trash can icon associated with the user.
  2. Click OK to confirm the deletion.
