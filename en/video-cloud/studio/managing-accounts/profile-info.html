Managing Your Profile

Video Cloud Studio users can use the Profile Information page in the Studio to view and change their own personal account information.

To access the Profile Information page:

  1. Sign in to Video Cloud Studio.
  2. On the Studio home page, click the user name - account area and then the My Profile link.

Editing profile information

Any of the fields in the Profile Information section can be edited.

Notes on editing:

  • You can change your first name, last name and the e-mail address you use to log in to the Studio
  • If you change your e-mail address (which is the user name you use to sign in with), you will be signed out of the Studio and will need to sign in again with your new e-mail address
  • You can change your password but you must verify your current password
  • Passwords need to be no shorter than 8 letters and contain at least one number and one letter
  • You can use the Language dropdown to select the Video Cloud Studio language (currently only English, Spanish and Japanese are available)

Note: If you are an Administrator, you can use the Managing Users page in Video Cloud Studio to change a user's role.

Click Save Changes to save the changes.
