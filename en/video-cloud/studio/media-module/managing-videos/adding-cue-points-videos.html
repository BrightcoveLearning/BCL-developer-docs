Adding Cue Points to Videos

In this topic you will learn about cue points and how to add them to your videos using the Media module.

A cue point is a marker at a precise time point in the duration of a video. You can use cue points to trigger mid-roll ads (that is, ads that occur during playback of the video content, like a commercial break) or to separate chapters or scenes in a long-form video.

There are two types of cue points:

  • Ad - Used to trigger ad requests
  • Code - Trigger an event that you can listen for and respond to using code; often used to indicate a chapter or scene break in a video

Cue points can be added to videos using the Video Cloud Media module or by using the Catalog API. For more information on adding cue points using the Catalog API, see the developer document Using Cue Points.

This topic is organized into the following sections:

  • Adding cue points
  • Editing cue points
  • Deleting cue points

Adding cue points

To add cue points to a video, open the Media module and follow these steps:

  1. Click the video title link to open the video properties page.
  2. Use the preview player to play the video. Pause the video at the point where you want to add a cue point.
  3. Click Add Cue Point.
  4. If needed, adjust the timecode hh:mm:ss values for the cue point.
  5. Select the type of cue point, code or ad.
  6. Enter a Name and other Metadata for the cue point.

Editing cue points

To edit cue points in a video, open the Media module and follow these steps:

  1. Click the video title link to open the video properties page.
  2. Click the cue point to edit in the timeline.

    Note: Once in edit mode, the Previous and Next buttons can be used to navigate to other cue points.

  3. Edit the cue point properties and then click Save.

Note: It is not possible to have multiple cue points with the same timecode.

Deleting cue points

To delete cue points, open the Media module and follow these steps:

  1. Click the video title link to open the video properties page.
  2. Click the cue point to delete in the timeline.
  3. Click the Delete icon ()next to the cue point properties.
  4. Confirm the deletion.
