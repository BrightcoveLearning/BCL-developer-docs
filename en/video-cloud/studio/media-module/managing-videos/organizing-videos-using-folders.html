Organizing Videos Using Folders

In this topic you will learn how to organize videos using folders inside the Media module.

Folders can be used to organize videos inside of the Media module. Folders can be used to organize videos by subject, author, product, region, etc. Having videos in folders makes it easier to manage a Video Cloud account with hundreds or thousands of videos.

Note: Nested folders are not supported.

This topic is organized into the following sections:

  • Creating folders
  • Adding videos to folders
  • Removing videos from folders
  • Deleting and renaming folders

Creating folders

To create a new folder, follow these steps:

  1. Access the side navigation panel by clicking the side navigation button .
  2. Click the New Folder link.
  3. Enter a name for the folder using the following guidelines. Press enter.
    • Name is limited to 100 characters
    • Folder names must be unique within an account
    • Illegal character: /
  4. Close the side navigation panel by clicking the side navigation button .

Adding videos to folders

Videos can be added to folders using the following methods:

  • Editing the video properties

Editing the video properties

Videos can be added to a folder by editing the video properties and assigning a folder in the Video Properties panel.

  1. In the Media module, click the link for the video.
  2. In the Video Information panel, click Edit.
  3. Select a Folder and click Save.

Removing videos from folders

Videos can be removed from folders using the following methods:

  • Editing the video properties

Editing the video properties

Videos can be removed from a folder by editing the video properties and removing the folder in the Video Information panel.

  1. In the Media module, click the link for the video.
  2. In the Video Information panel, click Edit.
  3. Set the Folder to --- and click Save.

Deleting and renaming folders

Folders can be renamed or deleted by following these steps:

Note: If you try to delete a folder containing videos, an error message will be displayed.

  1. Click the side navigation button to access the side navigation panel.
  2. Click the folder to be renamed or deleted. Click the down arrow () next to the folder and click Rename or Delete.
  3. Close the side navigation panel by clicking the side navigation button.
