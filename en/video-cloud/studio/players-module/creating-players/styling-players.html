Styling Players

This topic provides an overview of the player styling settings that can be configured.

Configuring style settings

To configure the player style settings, follow these steps:

  1. Click Players in the navigation header to open the Players module.
  2. Click the link for the player you want to configure.
  3. Locate the Styling section and click Edit.

The following player style settings can be configured:

  • Icons
  • Progress Bar
  • Background

After any changes have been made, make sure to Save the changes and republish the player for the changes to be visible.

Icons

The Icons setting controls the player icon colors, this includes the play/pause buttons, elapsed time and the volume and full screen icons.

To change the icon color:

  1. Click the Icon color picker.
  2. Click a color on the ride side of the control.
  3. Click in the square to select a shade of the desired color. A color can also be set by entering a hex color code into the text control.
  4. The slide below the color square can be used to set the opacity of the controls.
  5. Click outside of the control to modify another color property or click Save to save the changes.

Progress Bar

The Progress Bar setting controls the progress bar color.

To change the progress bar color:

  1. Click the Progress Bar color picker.
  2. Click a color on the ride side of the control.
  3. Click in the square to select a shade of the desired color. A color can also be set by entering a hex color code into the text control.
  4. The slide below the color square can be used to set the opacity of the progress bar.
  5. Click outside of the control to modify another color property or click Save to save the changes.

Background

The Background setting controls the background color of the player controls and the play button.

To change the background color:

  1. Click the Background color picker.
  2. Click a color on the ride side of the control.
  3. Click in the square to select a shade of the desired color. A color can also be set by entering a hex color code into the text control.
  4. The slide below the color square can be used to set the opacity of the background color.
  5. Click outside of the control to modify another color property or click Save to save the changes.
