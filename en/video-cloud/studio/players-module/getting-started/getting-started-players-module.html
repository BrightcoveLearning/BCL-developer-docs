Getting Started With the Players Module

This topic provides an overview of the Players module which is used to used to create and manage video players.

If you are looking to quickly get started creating and publishing a player, check out the Quick Start: Using the Players Module to Create and Publish a Player topic.

Using the Players module

To access the Players module, login to Video Cloud Studio and click Players in the navigation header.

The body of the page will display the player name, ID, modified date and player status. The status icon represents one of the following:

  • - Indicates that changes have been made to the preview player but the changes have not been published
  • - Indicates that the player is activate and all changes made to the player have been published
  • - Indicates that the player is inactive

Using the Players module, you can:

  • Create a new player
  • Edit player properties
  • Duplicate a player
  • Delete a player
  • Generate the embed code and URL for a player

Creating a new player

To create a new player using the Players module:

  1. Click the New Player button.
  2. Enter a name and short description for the player and then click Save.
  3. The new player will appear in the list of players.

Editing player properties

There are numerous player properties that can be configured to customize the look and feel of the player:

  • General player metadata
  • Styling
  • Player settings
  • Plugins
  • Social Sharing
  • Endscreen

For information on configuring player properties, see Setting Player Properties.

Note: Any changes made to a player may take up to 5 minutes to appear in published players.

Duplicating a player

Duplicating a player allows you to create a new player based upon an existing player. To duplicate a player in the Players module:

  1. Check the box next to the player you want to duplicate.
  2. Click the Duplicate button.
  3. Enter a Name and Short Description for the player.
  4. Click Save.

Deleting a player

Deleting a player removes it from your account and cannot be undone. To delete a player in the Players module:

  1. Check the box next to the player you want to delete.
  2. Click the Delete button.
  3. Click Delete on the delete player dialog to confirm the deletion.

Generate the embed code

The Embed Code & URL button can generate the player embed code and URL for the preview player or the published player. Typically, the preview player embed code and URL are used to preview the changes being made to a player. This gives you the ability to see the changes in action before publishing the player out to a production site. Once a player is published, changes will be reflected in all instances of that particular player.

For information on generating the publishing code for a player, see Generating Player Publishing Code.
