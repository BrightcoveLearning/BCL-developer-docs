Comparing Player Publishing Code

This topic covers the differences between the player publishing code for Smart Players and the new Brightcove Player.

The Players module provides the ability to generate the player publishing code for Brightcove players. To learn more about publishing Brightcove players and the differences between the types of publishing code that can be generated, see Generating Player Publishing Code.

This topic is broken up into the following sections:

  • JavaScript embed code
  • iFrame embed code

JavaScript embed code

Sample JavaScript publishing code for the Smart Player is shown below.

<script language="JavaScript" type="text/javascript" src="http://admin.brightcove.com/js/BrightcoveExperiences.js"></script>
<object id="myExperience" class="BrightcoveExperience">
   <param name="bgcolor" value="#FFFFFF" />
   <param name="width" value="480" />
   <param name="height" value="270" />
   <param name="playerID" value="1995095325001" />
   <param name="playerKey" value="AQ~~,AAAB0FLIj4k~,G3LRwE3nWw2XayhKaUy05iaiYJe86deo" />
   <param name="isVid" value="true" />
    <param name="isUI" value="true" />
    <param name="dynamicStreaming" value="true" />
    <param name="@videoPlayer" value="3776275116001" />
</object>
<script type="text/javascript">brightcove.createExperiences();</script>

Sample JavaScript publishing code for the new Brightcove Player is shown below.

<video width=”480”
       height=”270”
       data-account="1305620570001"
       data-player="3ad80e9d-67fa-4536-9fc8-6250b64446e4"
       data-embed="default"
       data-video-id="3528603921001"
       class="video-js" controls>
</video>
<script src="//players.brightcove.net/1305620570001/3ad80e9d-67fa-4536-9fc8-6250b64446e4_default/index.min.js"></script>

iFrame embed code

While there is no option to generate iFrame embed code in Video Cloud Studio, it is possible to embed the URL for a player into an iFrame. A Sample Smart Player URL is shown below.

http://link.brightcove.com/services/player/bcpid3577739224001?bckey=AQ~~,AAAAAFigZAk~,3mLHk34QtPckcJBD6fbG15ynasxNERfY&bctid=3738138354001

Sample iFrame embed code for the new Brightcove Player is shown below.

<iframe width=”480”
        height=”270”
        src='//players.brightcove.net/1305620570001/3ad80e9d-67fa-4536-9fc8-6250b64446e4_default/index.html'
        allowfullscreen
        webkitallowfullscreen
        mozallowfullscreen>
</iframe>

Video ID - You can control which video the player will load by modifying the data-video-id property of the video tag.

Styles - All style settings are controlled via CSS or standard HTML5 video tag properties (e.g., “controls”).

Player Key - The player key for the new Brightcove Player is embedded within the player itself.
