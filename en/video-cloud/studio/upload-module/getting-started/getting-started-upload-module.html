Uploading Videos Using the Upload Module

This topic provides an overview of the Upload module which is used to used to upload videos to a Video Cloud account.

Note: The New Video Cloud Studio is currently in beta and is not supported by Brightcove Support. If you would like access to the beta, please contact your Brightcove Account Manager.

The Upload module uses a dynamic ingest process to retrieve and transcode video content. When you select a file to upload, Video Cloud uses a pull-based process to pull the content from a storage location where it then uses the Zencoder transcoding service to transcode the content.

Using the Upload module

To access the Upload module, login to Video Cloud Studio and click Upload in the navigation header.

The Upload module can also be accessed at: https://studio.brightcove.com/products/videocloud-pre-release/upload

This topic contains the following sections:

  • Uploading Videos
  • Viewing and Editing video properties

Uploading Videos

Start by using the Select a Transcode Profile dropdown list to select an ingest profile.

The ingest profile contains the transcode settings that will be used when the video is transcoded. Note that the video renditions that are created are based upon the ingest profile that is selected NOT the transcode settings that are configured as part of your Video Cloud account settings (in Administration -> Transcode Settings). For specific details on the Brightcove supplied ingest profiles, see the Standard Ingest Profiles document. To have additional profiles added, or to edit an existing profile, contact your Brightcove Account Manager.

Note: The ability to create/edit/delete profiles will be available in a future release.

To select files to upload, the following options are available:

  • Drag and Drop - Drag files from the file system to the Upload module page. The entire page below the navigation header becomes the drop zone. When the file is dropped, the upload process will begin.
  • File selection dialog - Click the Browse for Videos button to open a file selection dialog and then browse and select files to upload.
  • Remote URL - Click the Enter URL button to specify the remote file location to upload from. File locations may include a public DropBox folder or Amazon S3 storage location. Click Add to Queue when finished.

Each file will display the file status:

  • Waiting - The file is waiting to be uploaded
  • Uploading - The file is being uploaded
  • Processing - The file is being transcoded
  • Processing Complete - The file transcoding is complete
  • Cancelled - The file upload was cancelled

Note that it is possible to change the ingest profile associated with a file upload as long as the file status is Waiting or Uploading. Once the file status has changed to Processing, the transcoding has begun and the ingest profile cannot be changed. To select a new ingest profile, click the Edit button associated with the video and then select a new profile.

Viewing and editing video properties

Click the Edit button associated with a video to view the video properties. Note that not all of the properties can be edited.

The following properties will be displayed:

  • Video Name
  • Video ID
  • File Status
  • Ingest profile used when transcoding
  • Video Title (editable and required)
  • Short Description (editable)
  • Tags (editable)
  • Reference ID (editable)

Click Save to save the changes. To learn more about video properties, see Editing Basic Properties.
