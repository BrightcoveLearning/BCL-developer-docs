generate_md5_checksum

:Boolean permalink

API Versions: V2

Default: false

Valid Values: true or false

Compatible Job Types: All

Description:

Generates an MD5 checksum for the generated output file.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "generate_md5_checksum": true
    }
  ]
}
      
<api_request>
  <input>s3://zencodertesting/test.mov</input>
  <outputs type="array">
    <output>
      <generate_md5_checksum type="boolean">true</generate_md5_checksum>
    </output>
  </outputs>
</api_request>
