live_stream

:Boolean permalink

API Versions: V2

Default: false

Valid Values: true or false

Compatible Job Types: live

Description:

Specifies that a job will have an RTMP streaming input. Returns a URL and stream name that will receive the input stream.

See our Live streaming guide for more information.

{
  "live_stream": true
}
      
<api_request>
  <live-stream type="boolean">true</live-stream>
</api_request>
