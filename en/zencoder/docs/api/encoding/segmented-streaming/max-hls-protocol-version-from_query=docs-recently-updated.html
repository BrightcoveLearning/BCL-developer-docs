max_hls_protocol_version

:Integer permalink

API Versions: V1, V2

Parent: outputs / streams

Default: 3 for Live outputs, 5 for sample AES encryption; otherwise, 2.

Valid Values: 2-5

Compatible Job Types: All

Example: 4

Description:

Sets the maximum HLS protocol version to use. Special features will be used as available.

Note: Only applies to HTTP Live Streaming outputs.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "type": "segmented",
      "max_hls_protocol_version": 4
    }
  ]
}
      
<api_request>
  <input>s3://zencodertesting/test.mov</input>
  <outputs type="array">
    <output>
      <type>segmented</type>
      <max_hls_protocol_version type="integer">4</max_hls_protocol_version>
    </output>
  </outputs>
</api_request>
