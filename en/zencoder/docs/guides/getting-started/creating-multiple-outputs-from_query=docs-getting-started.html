permalink Creating Multiple Outputs

You can specify multiple outputs by passing more than one hash in the outputs array, like this:

{
  "api_key": "93h630j1dsyshjef620qlkavnmzui3",
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}
