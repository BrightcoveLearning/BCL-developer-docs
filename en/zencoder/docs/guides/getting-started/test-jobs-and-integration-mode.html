permalink Test Jobs and Integration Mode

Test jobs are free on Zencoder. There are some limitations on the outputs created as test jobs.

Test jobs are:

  • 5 seconds long
  • Watermarked
  • Limited to one job/output processing at a time
  • Limited to 100 queued jobs
  • Limited to 50kb for transfer-only outputs

Jobs will be run as tests if your account is in Integration Mode, or if test is set to true in the API request.

New accounts start in integration mode.
